# E-Commerce Application 

*(in production)*

## **Stack Used**

   **Frontend**<br>
      Framework: React<br>
      State Management: Redux Toolkit<br>
      CSS: Vanilla CSS<br><br>
   **Backend**<br>
      Language: NodeJS<br>
      Server: ExpressJS<br>
      Sessioning: Express-Session<br>
      Auth: bCrypt<br>

To test the app, go to /views/ and /server/ and use npm install in both.
