# E-Commerce Application 

*(in production)*
<br>

## **Stack**

   **Frontend**<br>
      Framework: React<br>
      State Management: Redux Toolkit<br>
      CSS: Vanilla CSS<br><br>
   **Backend**<br>
      Language: NodeJS<br>
      Server: ExpressJS<br>
      Sessioning: Express-Session<br>
      Auth: bCrypt<br><br>
